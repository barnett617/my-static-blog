python安装web.py

### 方式一

下载python文件

```
wget http://webpy.org/static/web.py-0.37.tar.gz
```

最新开发版:https://github.com/webpy/webpy/tarball/master

解压

```
tar -zxvf web.py-0.37.tar.gz
```

安装

```
python setup.py install
```
